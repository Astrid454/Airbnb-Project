# Airbnb Web Platform

**Technologies:** HTML, CSS, SQL

**Description:**  
Developed a marketplace web platform that allows users to register as **Host** or **Guest**, publish listings, and manage reservations.  

**Features:**  
- User registration and authentication for Hosts and Guests  
- Hosts can create, update, and delete listings  
- Guests can browse listings and make reservations  
- SQL database for storing user and listing data  
- Clean, responsive web interface
